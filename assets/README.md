# Assets Directory

This directory contains static assets used by the application.

## Structure

- `images/` - Icons, logos, diagrams
- `templates/` - Email templates, document templates
- `prompts/` - LLM prompt templates
- `config/` - Configuration files and schemas

## Usage

Store static resources that are part of the application distribution here.

**Note**: This directory is gitignored except for this README and .gitkeep files.
